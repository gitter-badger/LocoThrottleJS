Prototypehelp
inputLoad json file
Help | Prototypeclose

When you select this file, you're telling the physics engine how to make your locomotive behave. Prototype.json files should be located in the /prototype folder in your locoThrottle.js install directory.

Unfortunately, at this point in time, there may not be a prototype.json file for your locomotive. If that is the case, I highly suggest you write one. It is not hard, you don't have to know any kind of programming, and it benefits everyone.
