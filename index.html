What is locoThrottle.js?

locoThrottle.js is a software for realistic control of model trains. It runs in a web browser on almost any device, and works with any layout that has JMRI.

  • helpHow do I set it up?

    Right now, you shouldn't try locoThrottle.js unless you're a developer. It is unfinished, with many new features being untested and potentially unstable. This may cause problems/mess things up while you're using it, and that would be bad. Therefore, be aware that this software is provided on a use-at-your-own-risk basis right now. If you want to download it as a standalone program, you will need a web server to host it on. The easiest way to download the program is to visit our GitHub page and click "Download Zip". Then, extract the contents of the ZIP to your web server, and you're good to go.

    If you want to contribute to the development, your help is greatly appreciated! You should clone out Git repo instead of just downloading the ZIP, because this will make it easy for you to make changes. You can visit our GitHub page to start helping.
